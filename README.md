# Async File Downloader — C#

Downloads multiple files simultaneously using async/await and Task.WhenAll.

## Concepts covered
- async / await — non-blocking I/O
- Task and Task<T> — representing async work
- Task.WhenAll — parallel execution
- CancellationToken — cooperative cancellation
- HttpClient — making HTTP requests
- Exception handling in async code (try/catch/finally)

## How to run
Open in Visual Studio 2022 and press F5.
Files are saved to bin/Debug/netX.X/Downloads/
